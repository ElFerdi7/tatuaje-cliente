Bienvenidos al proyecto H&T STUDIO

El proyecto de referencia en la gestión de un estudio de tatuajes.

¿Qué es H&T STUDIO?

  • H&T STUDIO es una aplicación con la que podrás gestionar tu estudio de tatuaje.
  • Esta aplicación web es además un proyecto para el ciclo superior de Desarrollo de Aplicaciones Web (DAW)

¿Quién impulsa este proyecto?

  • El autor de la aplicación base es Fernando Martin (fmv_7791@hotmail.com); alumno de segundo del Ciclo Formativo de Grado Superior denominado Desarrollo de Aplicaciones Web.
  • La aplicación ha sido desarrollada mediante el uso de los conocimientos y programas hechos en clase.
  • El programa esta creado a partir de las indicaciones de un cliente concreto para el futuro uso de la aplicación.

¿Qué se pretende con este proyecto?

  • Realizar una aplicación usable para el futuro.
  • Participar de manera práctica en la creación de una aplicación web compleja
